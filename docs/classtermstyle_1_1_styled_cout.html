Detailed Description

A class that provides styled output to the standard output stream.

The StyledCout class allows you to output text with custom styles to the standard output stream (std::cout). It takes a PresetConfig object as a parameter, which contains the configuration for the styles to be applied. The styles are applied to the text when it is printed to the standard output stream.
